# FELIX vs Gaussian Pearson Correlation Coefficient Plotting Scripts

## Overview
The `FELIX_Gaussian_PCC_Plot.m` script is designed to facilitate the comparison of IRMPD experimental data with Gaussian calculated spectra. This allows researchers to visually assess the agreement between their experimental results and theoretical predictions.

## Requirements
Before using the script, ensure you have the following:
- MATLAB installed on your system.
- The necessary experimental data files in the appropriate format.

## Usage Instructions
1. **Prepare Your Data**: 
   - Ensure your IRMPD experimental data is available in a format that the script can read. This typically includes properly formatted spectra files.
   
2. **Setup the Script**:
   - Open MATLAB and navigate to the directory where `FELIX_Gaussian_PCC_Plot.m` is located.
   
3. **Run the Script**:
   - Execute the script by typing `FELIX_Gaussian_PCC_Plot` in the MATLAB command window and pressing Enter.
   
4. **Input Data**: 
   - Follow the prompts to input the file paths for your experimental data and Gaussian calculated spectra. Make sure to provide the full path where your files are located.
   
5. **Visualize the Comparison**:
   - The script will generate plots comparing your IRMPD data with the Gaussian calculated spectra. Analyze the graphical output to understand the correlation between the experimental results and theoretical models.

## Conclusion
By following these instructions, you will be able to utilize the `FELIX_Gaussian_PCC_Plot.m` script effectively to compare and analyze your spectroscopic data.
