# IRMPD Gaussian Comparison Plot

## Instructions for Use

1. **Prerequisites**: Ensure that you have Python installed on your system along with the following libraries:
   - numpy
   - matplotlib
   - pandas

2. **Usage**:
   - Clone the repository to your local machine:
     ```bash
     git clone https://github.com/wsl530-tpp/IRMPD-Gaussian-Comparison-Plot.git
     cd IRMPD-Gaussian-Comparison-Plot
     ```
   - Run the script using the following command:
     ```bash
     python irmpd_gaussian_comparison.py <input_file>
     ```
   - Replace `<input_file>` with the path to your input data file.

3. **Output**: The script generates a plot comparing the IRMPD data with Gaussian fits and saves it as `comparison_plot.png` in the current directory.

4. **Customization**: You can customize the plot appearance by editing the script parameters.

5. **Support**: For any issues, please open an issue in this repository.
